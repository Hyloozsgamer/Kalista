## ⚙️ 1. Martial Poise (Passive) Analytics: Attack Speed vs Jump Cadence

Unlike standard ADCs, Kalista's DPS is intrinsically tied to her movement (Jump). There is a hard cap where additional Attack Speed (AS) becomes inefficient if you hop on every auto.

| AS Value | Jump Efficiency | Tactical Implication | LCK Notes |
| :---: | :---: | :--- | :--- |
| **0.65 - 1.25** | `100%` Sync | Early-to-Mid Game | Perfect synergy. Every attack generates a hop without cutting wind-up. |
| **1.30 - 1.60** | `90%` Sync | Mid-Game Core | Frame-perfect clicks required. Jumping backwards is slower than jumping forwards. |
| **1.70 - 2.00** | `70%` Drift | Late Game | **WARNING:** Jumping reduces DPS by slowing auto-cadence. You MUST stand still occasionally to maximize DPS. |
| **2.00+** | `50%` Cap | Overcapped | Do not build more Attack Speed items beyond 2.0. Focus on AD / Penetration. |

> [!CAUTION]
> **Boot Progression Alert (Tier Upgrades):**
> Kalista's jump distance scales exclusively with BOOT TIER. Rushing `<img src="https://ddragon.leagueoflegends.com/cdn/14.5.1/img/item/3006.png" width="16"> Berserker's Greaves` is mathematically optimal. It increases jump distance by **+25%**, making it the single largest power spike in her early game.

## 📈 2. Rend (E) Damage Threshold Matrix

Rend execution requires precise mental calculation. The base damage plus scaling per spear determines the kill threshold.

`Formula: Base Damage (20-60) + [Per Spear Damage (10-34)] + %AD scaling`

### First Blood Thresholds (Levels 1 - 3)
| Scenario | Spears Needed | Execution Health Threshold | Setup Concept |
| :--- | :---: | :---: | :--- |
| **Level 1 Trade** | 3 Stacks | `110 HP` | Auto x2 -> Back away -> E to slow and disengage. |
| **Level 2 All-in** | 5 Stacks | `~210 HP` | `Auto x4 -> Q (Pierce) -> E (Rend) mid Q animation.` |
| **Level 3 Engage** | 7+ Stacks | `~340 HP` | Support CC -> Full stack rotation -> E execution. |

### Epic Objective Securing (Baron / Dragon)
Kalista replaces Jungler Smite in reliability if stacked properly.
* **Baron (Late Game):** ~40 Stacks = `~2500 - 3000 Damage`. Surpasses Smite (1200).
* **Execution Rule:** Rend + Smite perfectly simultaneously to secure objectives instantly.
